## Maryland DLLR Search Tips

### Form Fields
- **Last Name:** Required field
- **First Name:** Optional but recommended for accuracy
- **CAPTCHA:** Must be completed manually (reCAPTCHA v2)

### Search Results
- Results show: Name, License Number, License Type, Status, Expiration Date
- Check for exact name matches
- Verify license type is Real Estate related

### Common License Types in Maryland
- Real Estate Broker
- Real Estate Salesperson
- Property Manager (if separate license type)
