## File Locations

### Evidence Files
- Source: `evidence/` (organized by type)
- Extracted: `research/pdf_evidence_extracted.json`, `research/excel_evidence_extracted.json`
- Summary: `research/pdf_evidence_summary.csv`, `research/excel_evidence_summary.csv`

### Filing Materials
- Checklist: `filings/filing_checklist.csv`
- Evidence Package: `filings/filing_evidence_package.json`
- Executive Summary: `filings/executive_summary.md`
- Agency Registry: `research/regulatory_agencies_registry.json`

### Analysis Outputs
- Reports: `outputs/reports/`
- Summaries: `outputs/summaries/`
- Research: `research/`

### Scripts
- Extraction: `scripts/extraction/`
- Analysis: `scripts/analysis/`
- Search: `scripts/search/`
- Legacy: `scripts/legacy/`
