# Data Directory

All data files used and generated by the **AI-Powered** Kettler Data Analysis project with advanced machine learning capabilities.

## Directory Structure

```
data/
├── source/          # Source data (authoritative)
├── scraped/         # Scraped data
├── raw/             # Raw search results (gitignored)
├── cleaned/         # Cleaned data (gitignored)
├── vectors/         # Vector embeddings
├── schema.json      # Complete schema (FK/PK)
├── metadata.json    # Global metadata
├── DATA_DICTIONARY.md
├── ONTOLOGY.md
└── ANCESTRY.md
```

## Source Data

### Firms (`source/skidmore_all_firms_complete.json`)
- **Primary Key:** `firm_license` (10-digit)
- **Records:** 38 firms
- **Key Fields:** firm_name, address, state, principal_broker, dates

### Individual Licenses (`source/skidmore_individual_licenses.json`)
- **Primary Key:** `license_number` (10-digit)
- **Records:** 40+ licenses
- **Key Fields:** name, address, state, license_type

## Schema

All data follows normalized schema with **Primary Keys (PK)** and **Foreign Keys (FK)**.

**Quick Reference:**

| Table | PK | Key FK | File |
|-------|----|--------|------|
| firms | `firm_license` | `individual_license` → `individual_licenses.license_number` | `source/skidmore_all_firms_complete.json` |
| individual_licenses | `license_number` | None | `source/skidmore_individual_licenses.json` |
