## Structure Summary

### Top-Level Organization

1. **00_Administration** - Master index, templates, reference materials
2. **01_Federal** - Federal agency filings (HUD, DOJ, FTC, CFPB, SEC, IRS)
3. **02-11_State** - State-level filings (VA, MD, DC, NC, FL, TX, GA, PA, NJ, DE)
4. **12-14_Local** - Local county/city filings
5. **15_Professional_Organizations** - Professional association complaints
6. **16_Evidence_Repository** - Comprehensive evidence collection
7. **17_Analysis_Research** - Research and analysis materials
8. **data/** - Data files and documents

### Key Features

- **Hierarchical organization** by jurisdiction and agency type
- **Comprehensive evidence** organized by category
- **Individual complaints** separated by respondent
- **Correspondence and exhibits** organized per filing
- **Multi-state coverage** across 10+ states
- **Federal and local** coverage

### File Types

- PDF documents (complaints, correspondence, exhibits)
- Google Docs/Sheets (editable documents)
- Images (screenshots, evidence)
- Excel files (billing history, data)
- Markdown files (documentation)
